## 📖 Proje Hakkında
**Siber Savaşçılar**, web uygulamaları ve ağ sistemleri için otomatik sızma testleri gerçekleştirmek amacıyla geliştirilmiş akademik bir güvenlik aracıdır. Zafiyet taraması, güvenlik açığı analizi ve raporlama özelliklerini tek bir merkezde toplar. Dijital varlıkları daha güvenli hale getirmek amacıyla, OWASP standartları referans alınarak tasarlanmıştır.

## ✨ Temel Özellikler
- 🎯 **Otomatik Zafiyet Tarama Motoru:** Ağ ve web tabanlı açıkların tespiti için temel tarama yetenekleri.
- 📊 **Raporlama Modülü:** Karmaşık tarama sonuçlarını anlaşılır formatlara dönüştürme.
- ⚙️ **Özelleştirilebilir Senaryolar:** İhtiyaca göre esnetilebilen test profilleri.
- 🖥️ **Web Tabanlı Yönetim Paneli:** Sürecin, sonuçların ve görevlerin kolayca izlenebileceği kullanıcı dostu arayüz.
