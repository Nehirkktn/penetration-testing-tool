## 📖 Proje Hakkında
**Siber Savaşçılar**, web uygulamaları ve ağ sistemleri için otomatik sızma testleri gerçekleştiren kapsamlı bir güvenlik aracıdır. Zafiyet taraması, güvenlik açığı analizi ve raporlama özelliklerini tek bir merkezde toplar. Dijital varlıkları daha güvenli hale getirmek amacıyla, OWASP standartlarına tam uyumlu test senaryoları ile donatılmıştır.

## ✨ Temel Özellikler
- 🎯 **Otomatik Zafiyet Tarama Motoru:** Ağ ve web tabanlı açıkların hızlı tespiti.
- 📊 **Gelişmiş Raporlama Modülü:** Karmaşık tarama sonuçlarını anlaşılır, yönetici dostu formatlara dönüştürme.
- ⚙️ **Özelleştirilebilir Senaryolar:** İhtiyaca göre esnetilebilen, dinamik test profilleri.
- 🖥️ **Web Tabanlı Yönetim Paneli:** Sürecin, sonuçların ve görevlerin kolayca izlenebileceği kullanıcı dostu arayüz.
